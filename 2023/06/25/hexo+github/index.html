三、通过ssh连接将博客部署到github

静态页面（本地博客）已经生成，还需要将其部署到GitHub上。一来搭建的博客别人可以（以一个链接）访问到，二来需要有个托管平台，这样可以关注博客内容本身而不是麻烦的管理。
